# Proyecto-Estructura-de-Datos
Proyecto del curso Organización de Archivos y Estructuras de Datos
Marvin Andres Angulo Atencio
ISB-08
Sistema desarrollado en C++ que implementa múltiples estructuras de datos para la gestión de contactos.  
El programa permite agregar, buscar, ordenar, eliminar y organizar contactos utilizando listas, colas, pilas y árboles binarios.

Funcionalidades principales:
1.	Módulo de Contactos:
- Agregar contactos
- Listar contactos
- Ordenar (Nombre / Teléfono)
- Búsquedas (Secuencial y Binaria)
2.	Eliminación con Lista Enlazada:
- Copia de contactos a una lista enlazada
- Eliminación usando nodos enlazados
3.	Historial de Operaciones (Pila):
- Registro manual de operaciones 
4.	Cola de Contactos Pendientes:
- Encolar contactos pendientes de agregar
- Transferir contactos desde la cola hacia la agenda
5.	Árbol Binario de Búsqueda:
- Construcción del árbol por nombre
- Búsqueda por nombre
- Recorridos inorden y  preorden
- Visualización del árbol

Lenguaje: c++
